# Race Hub v2 Admin Workflow

## Registration workflow

1. Driver/team submits public form.
2. Race Hub creates a pending JSON submission.
3. SRCS admin reviews submission.
4. Admin contacts driver/team if needed.
5. Admin approves, rejects, or leaves pending.
6. Approved entries are added to official event data.
7. Public pages read only approved entries.

## Why this matters

Public input should never directly change official standings, classifications, or confirmed event entries.

This keeps SRCS data clean as the system scales to:

- 2026 championship
- Spa 3-hour Endurance
- Rookie intake
- League grids
- Challenger
- Premier
- Reserve drivers
